## 🎬 直接上传的步骤

### 步骤 1️⃣：打开你的 GitHub 仓库

```
https://github.com/你的用户名/kuhnpohl-media
```

### 步骤 2️⃣：点击 "Add file"

```
看右侧按钮，有两个选项：
- Add file
- Upload files

点击 "Upload files"
```

### 步骤 3️⃣：拖拽或选择文件

```
方法 A：拖拽
  从电脑直接拖拽文件到 GitHub 页面

方法 B：点击选择
  点击 "choose your files"
  从 /mnt/user-data/outputs 选择文件
```

### 步骤 4️⃣：一次上传所有文件

```
GitHub 可以同时上传多个文件！

选择这些文件：
  □ 所有 .webp 图片
  □ 所有 .mp4 视频
  □ 所有 .md 文档
  □ 所有 .html 文件

一起拖拽进去，一次性上传！
```

### 步骤 5️⃣：点击 "Commit changes"

```
上传完后，点击提交
```

### ✅ 完成！

```
所有文件都在 GitHub 了！
```
