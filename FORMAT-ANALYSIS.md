## 🔧 如果需要转换 (参考)

### 转换 PNG 为 WebP

```bash
# 单个文件
convert image.png -quality 85 -define webp:method=6 image.webp

# 批量转换
for f in *.png; do
  convert "$f" -quality 85 -define webp:method=6 "${f%.png}.webp"
done
```

### 转换视频为 MP4

```bash
# 从任何格式转 MP4
ffmpeg -i video.mov -c:v libx264 -preset medium -crf 23 output.mp4

# 生成不同分辨率
ffmpeg -i video.mp4 -vf "scale=1280:720" -c:v libx264 -c:a aac video-720p.mp4
```
